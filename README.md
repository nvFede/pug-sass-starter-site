# Simple Starter Site Boilerplate

Simple starter site with:

- Webpack
- Pug
- PostCSS
- SASS
- Babel

## Download

```bash
git clone git@github.com:nvFede/pug-sass-starter-site.git
```

## Installation

```bash
cd pug-sass-starter-site
```

```bash
npm install
```

## Live Server

```bash
npm run start
```

## Build Production

```bash
npm run build
```
